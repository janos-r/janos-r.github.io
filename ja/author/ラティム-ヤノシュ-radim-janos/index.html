ラディム　ヤノシュ Radim Janoš

デベロッパー

プラハ、チェコ共和国

チェコ人の血と、北部チェコのドイツ人の家族の間で育ち、2016 年に経済学の修士を卒業し、2017 年に IT と、最新のテクノロジーに興味を持ってから、1.5 年を経て、現在の「Node.js」でのデベロッパーとしてのポジションを 2019 年から開始しました。新しいことを学び続け、バックエンドデベロッパーとしての道を楽しんでいます。最近は、社内で活動的に使うようになったクラウド(Azure, Docker, Kubernetes, Helm)に注力し始めました。

2018 年の初めに Linux Mint を試してみて、すぐに Windows から切り替えて、大きな Linux とフリーソフトウェアの提唱者になりました。

囲碁のゲームは、古くからの大きな趣味の内の 1 つです。地域の囲碁クラブで、生徒と管理をしたり、自分でウェブサイトを作ったり、4 年間もの間、年 1 回の全国のトーナメントを組織したりしました。
